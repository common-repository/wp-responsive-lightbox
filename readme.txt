Plugin for adding a customizable, responsive lightbox pop-up AD, banner or call of action on home page or any page of your liking.

== Description ==

- Responsive Pop Up Lightbox Ads & Banners
- URL link field
- Lightbox width
- Background color
- Add to Multiple Page/Post IDs
- More!

== Installation ==

1. Download Plugin
1. Upload the 'wp-responsive-lightbox' folder to the '/wp-content/plugins/' directory
1. To edit settings, click left menu 'settings" and then click 'responsive lightbox'.
